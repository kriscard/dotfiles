# SEO Checklist for Blog Posts

Quick pre-publish checklist for christophercardoso.dev posts.

## Before Writing

- [ ] **Primary keyword identified** (what people search for)
- [ ] **Search intent understood** (tutorial, comparison, explanation)
- [ ] **Target word count set** (based on competition and depth needed)

## Title & Meta

- [ ] **Title includes primary keyword** (preferably near the start)
- [ ] **Title is compelling** (would you click it?)
- [ ] **Title under 60 characters** (doesn't get cut off in search results)
- [ ] **Meta description written** (150-160 characters)
- [ ] **Meta description includes keyword** and a call to action

### Meta Description Template
```
Learn how to [action] with [technology].
This [tutorial/guide] covers [key points] with practical examples.
```

## URL & Structure

- [ ] **URL slug is short** and includes primary keyword
- [ ] **URL uses hyphens** (not underscores)
- [ ] **No dates in URL** (keeps content evergreen)

**Good:** `/blog/nextjs-app-router`
**Bad:** `/blog/2024/01/how-to-use-the-nextjs-app-router-complete-guide`

## Content Structure

- [ ] **Primary keyword in first paragraph**
- [ ] **H2 headings include secondary keywords** (at least 1-2)
- [ ] **Logical heading hierarchy** (H1 → H2 → H3, no skipping)
- [ ] **Short paragraphs** (2-4 sentences max for readability)
- [ ] **Bulleted/numbered lists** for scannable content

## Links

- [ ] **2-3 internal links** to other posts on your site
- [ ] **2-3 external links** to authoritative sources
- [ ] **Links use descriptive anchor text** (not "click here")
- [ ] **External links open in new tab** (if applicable)

## Images & Code

- [ ] **All images have alt text** (descriptive, includes keyword if natural)
- [ ] **Images are optimized** (compressed, proper dimensions)
- [ ] **Code blocks specify language** (for syntax highlighting)
- [ ] **Code examples are complete** and runnable

## Technical

- [ ] **Page loads fast** (no huge unoptimized images)
- [ ] **Mobile-friendly layout**
- [ ] **No broken links**
- [ ] **Canonical URL set** (if republishing elsewhere)

## Engagement Signals

- [ ] **Reading time displayed**
- [ ] **Share buttons work** (if applicable)
- [ ] **Comments enabled** (if applicable)
- [ ] **Related posts linked** at the end

## Final Checks

- [ ] **Read aloud** - does it flow naturally?
- [ ] **Spell check passed**
- [ ] **All code tested** and working
- [ ] **Published date correct**

## Quick Reference: Keyword Placement

| Location | Required? | Notes |
|----------|-----------|-------|
| Title | Yes | Near the start |
| First paragraph | Yes | Natural inclusion |
| At least one H2 | Yes | Don't force it |
| URL slug | Yes | Short and clean |
| Meta description | Yes | With call to action |
| Alt text | If natural | Don't keyword stuff |
| Throughout content | Naturally | 1-2% density max |

## SEO Don'ts

- Don't stuff keywords unnaturally
- Don't write for search engines over humans
- Don't use the same anchor text repeatedly
- Don't ignore meta descriptions
- Don't publish thin content (under 300 words for substantial topics)
- Don't forget to update old posts with internal links to new ones
